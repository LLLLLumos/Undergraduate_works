# JAVA教务管理系统
## 简介

实现了管理员端教务管理系统的功能，包括管理员登录，学生信息管理、课程信息管理、成绩管理的增删查改操作。连接mysql数据库实现数据的存储。

## 开发平台

- 编译器  MyEclipse10.7
- JDK 1.8.0_20
- 数据库 Mysql5.3
- 操作系统 Window 10
